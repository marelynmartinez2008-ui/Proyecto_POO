🐾 Sistema de Gestión - Centro de Adopción de Perritos

Este es un sistema web desarrollado con **Python** y **Flask** para gestionar la adopción de mascotas. Permite a los usuarios ver un catálogo de perros disponibles y a los administradores gestionar el historial de adopciones y añadir nuevos integrantes al sistema.

🚀 Características

- **Catálogo Dinámico:** Visualización de perritos con fotos, edad y raza.
- **Sistema de Usuarios:** Roles diferenciados para Adoptantes y Administradores.
- **Panel de Administrador:** - Visualización de historial de adopciones.
  - Formulario para agregar nuevos perros al catálogo.
- **Transacciones Seguras:** Registro de adopciones utilizando transacciones SQL (ACID) para garantizar la integridad de los datos.
- **Base de Datos:** Implementado con MariaDB/MySQL.

Tecnologías Utilizadas

* **Backend:** Python 3 + Flask
* **Base de Datos:** MariaDB / MySQL
* **Frontend:** HTML5, CSS3 (Google Fonts - Poppins)
* **Librerías:** `mariadb`, `flask`

Requisitos Previos

Antes de ejecutar el proyecto, asegúrate de tener instalado:
- Python 3.12+
- MariaDB Server
- Driver de MariaDB para Python: `pip install mariadb`
